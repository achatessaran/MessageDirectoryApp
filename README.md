# MessageDirectoryApp

## Overview

MessageDirectoryApp is a React Native mobile application developed using Expo and JavaScript. The application allows users to browse important message directories and view stored messages within each directory. The app was developed as part of Exercise 3 for the Mobile Programming course (COMP-5450-SA).

## Features

* Six message directories:

  * You
  * Home
  * Love
  * Family
  * Friends
  * School
* Colorful card-based user interface
* Directory icons and message counts
* View stored messages for each directory
* Back navigation to return to the home screen
* Search messages within a selected directory
* Responsive design for Android devices

## Technologies Used

* React Native
* Expo
* JavaScript
* Android Studio
* Expo Router

## Installation and Execution

### Prerequisites

* Node.js
* npm
* Android Studio
* Android Emulator (Pixel 6 recommended)

### Steps

1. Clone or download the project.

2. Navigate to the project directory:

```bash
cd MessageDirectoryApp
```

3. Install dependencies:

```bash
npm install
```

4. Start the Android emulator.

5. Run the application:

```bash
npm run android
```

The application will launch on the Android emulator.
