Overview
MessageDirectoryApp is a React Native mobile application developed using Expo and JavaScript. The application allows users to browse important message directories and view stored messages within each directory. 
Features
•    Six message directories:
1.     You
2.     Home
3.     Love
4.     Family
5.     Friends
6.     School
 
•    Colorful card-based user interface
•    Directory icons and message counts
•    View stored messages for each directory
•    Back navigation to return to the home screen
•    Search messages within a selected directory
•    Responsive design for Android devices
 
Technologies Used
•    React Native
•    Expo
•    JavaScript
•    Android Studio
•    Expo Router

Installation and Execution
Prerequisites:
•    Node.js
•    npm
•    Android Studio
•    Android Emulator (Pixel 6 recommended)
Steps:
1.        Clone or download the project.
2.     Navigate to the project directory where it was downloaded.
E.g., cd MessageDirectoryApp
3.     Install dependencies using the below command in Terminal->Bash
npm install
4.        Start the Android emulator (E.g., Pixel 6)
5.        Run the application using the below command in Terminal->Bash
npm run Android
The application will launch on the Android emulator.
